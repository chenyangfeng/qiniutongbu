**使用前请先配置空间信息！**

#### 配置说明：

+ `AccessKey`：AK
+ `SecretKey`：SK
+ `Bucket`：空间
+ `Domain`：域名value="http://obxxxx.bkt.clouddn.com/"
+ `Private`：是否私有
+ `Prefix`：资源前缀
+ `SaveAs`：本地保存路径 示例 value="D:\pic\"
+ `OverWrite`：覆盖本地文件

#### 项目依赖：

+ 依赖 `qiniutongbu.exe.config` 来记录配置信息.
+ 依赖 `LitJson.dll` 来处理 JSON，确保目录中含有该文件；
